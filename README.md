## Instalação

### Linux / OSX 
```bash
$ bash -c "$(curl -fsSL raw.github.com/brazilian-dev/dotfiles/master/bin/dotfiles)"
```

### Windows
```
cd .dotfiles
make.bat
```

## Uso

**Opções**

- `./dotfiles --help`: instruções de uso
- `./dotfiles --list`: lista com todos os aplicativos e dependências a serem instaladas
