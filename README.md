## How to run

```bash
$ sh install
```

## Instalation

```bash
$ curl -L raw.github.com/brazilian-dev/dotfiles/master/install | sh
# or
$ git clone https://github.com/brazilian-dev/dotfiles.git && cd dotfiles && sh install.sh
```
