## How to run

```bash
$ sh install
```

## Instalation

```bash
$ curl -L raw.github.com/org-moon-world/dotfiles/master/install | sh
# or
$ git clone https://github.com/org-moon-world/dotfiles.git && cd dotfiles && sh install.sh
```
